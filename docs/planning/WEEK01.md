# Week 1: Project Foundation & Architecture

## Overview
This first week is dedicated to setting up the core structure of the MTG DeckBuilding Application. The focus is on establishing a solid foundation for both the backend and frontend, ensuring the architecture is scalable and the database schema is well-defined.

## Weekly Goals
- [x] **Project Initialization**: Setup the basic web structure (HTML/CSS/JS folders) and Git configuration.
- [x] **Backend Architecture**: Configure the Spring Boot environment with Maven, including dependency management (Web, JPA, Security).
- [ ] **Database Design**: Define the initial entity relationships for MTG decks and cards.
- [ ] **Environment Setup**: Configure H2 in-memory database for rapid development and testing.

## Technical Details
- **Backend Framework**: Spring Boot 3.2.5
- **Persistence**: Spring Data JPA with H2 Database
- **Documentation**: Established `docs/planning` structure for project tracking.
