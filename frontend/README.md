# MTG DeckBuilding Platform - Frontend Mockups

Este directorio está destinado a almacenar las maquetas (mockups) temporales en HTML, CSS y JavaScript para visualizar cómo se vería la interfaz de usuario del backend antes de su migración final a React.

## Archivos recomendados para colocar aquí:
- `index.html`: Dashboard principal de la aplicación.
- `deckbuilder.html`: Creador de mazos interactivo.
- `styles.css`: Estilos globales y tema visual.
- `app.js`: Lógica de interacción local.
