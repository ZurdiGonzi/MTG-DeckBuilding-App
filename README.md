# MTG DeckBuilding App

## About the Author

I am a Software Engineering student, and this project is part of my professional portfolio to demonstrate my skills and development progress.

## Product Definition

This application is a specialized deck management tool for **Magic: The Gathering (MTG)**. It aims to provide a seamless experience for players to build, manage, and optimize their decks. Beyond standard features, the project will incorporate novel functionalities to enhance the user experience and provide unique value to the MTG community.
